线性离子阱中的时间晶体 (1)

模型的建立与有效哈密顿量的导出

我们考虑一个被限制在线性离子阱中的\(^{40}\mathrm{Ca}^{+}\)离子，该系统受到强度为\(\varepsilon\)的外部场驱动。在这一装置中，我们利用三束波长为729 nm的驻波激光模拟了多种耗散和增益过程。每束激光具有不同的作用：一束以速率\(\kappa\)引发线性阻尼，另一束以速率\(\lambda\)引入非线性阻尼，第三束则以速率\(g\)产生线性增益。

与本实验相关的能级结构如下图所示。\(^{40}\mathrm{Ca}^{+}\) 离子的 \(4\text{S}_{1/2}\) 态是其基态，而 \(3\text{D}_{5/2}\) 态可以通过 729 nm 激光束选择性地激发。通过调整三束激光的频率构建模型，其中两束激光分别失谐 \(\pm \omega _e\)，第三束激光失谐 $-2_e $，其中 $_e $ 为外加电场的频率。从 \(4\text{P}_{3/2}\) 态到 \(4\text{S}_{1/2}\) 态的跃迁以自发辐射为特征，其速率为 \(\gamma\)。此外，我们引入了一束 854 nm 的淬灭激光，用于快速清空 \(4\text{P}_{3/2}\) 态，将离子驱回基态，从而有效地最小化泄漏和去相干效应。

系统的哈密顿量为： \[\begin{align} \begin{aligned} H = & \omega_1 \left | 1 \right \rangle \left \langle 1 \right | + \omega_2 \left | 2 \right \rangle \left \langle 2 \right | + \omega_k a^\dagger a + \varepsilon(a \mathrm{e}^{-i \omega_e t} + h.c.)\\ & + \sum_{m = 1}^3 \frac{\Omega _m}{2}\mathrm{sin}[\eta(a + a^\dagger) + \phi_m]\left(\sigma_+\mathrm{e}^{-i \omega_L^m t} + h.c. \right )\\ & + \frac{\Omega_e}{2} \left( \left | 2 \right \rangle \left \langle 1 \right | \mathrm{e}^{-i \omega_L^e t} + h.c. \right ), \end{aligned} \end{align}\] 其中，参数\(\omega_1\)和\(\omega_2\)分别表示粒子处于量子态\(\left| 1 \right\rangle\)和\(\left| 2 \right\rangle\)时的能量本征值。\(\omega_k\)表示与离子阱腔模相关的轴向振动频率。符号\(\varepsilon\)表示外加电场的幅值。\(\Omega_m\)、\(\phi_m\)和\(\omega_L^m\)分别表示三束729 nm激光的拉比频率、相位偏移和频率。此外，\(\Omega_e\)和\(\omega_L^m\)表示854 nm激光的拉比频率和激光频率。\(\eta\)是\(^{40}\mathrm{Ca}^+\)离子的Lamb-Dicke参数。算符\(\sigma_+\)和\(\sigma_-\)定义为\(\sigma_+ = \left| 1 \right\rangle \left\langle 0 \right|\)和\(\sigma_- = \left| 0 \right\rangle \left\langle 1 \right|\)。然后，通过应用旋转算符，可以将哈密顿量转换到旋转参考系： \[\begin{align} \begin{aligned} \widetilde{H} = &-\Delta a^\dagger a + \varepsilon (a + a^\dagger)\\ &+ \sum_{m = 1}^3 \frac{\Omega _m}{2}\mathrm{sin}[\eta(a\mathrm{e}^{-i \omega_e t} + a^\dagger \mathrm{e}^{i \omega_e t}) + \phi_m]\left ( \sigma_+\mathrm{e}^{-i \delta_L^m t} + h.c. \right ) \\ & +\frac{\Omega _e}{2}\left [ \left| 2 \right\rangle \left\langle 1 \right| \mathrm{e}^{-i (\omega_L^e +\omega_1 - \omega_2)t} + h.c. \right ], \end{aligned} \end{align}\] 其中，\(\Delta = \omega_e - \omega_k\)，\(\delta_L^m = \omega_L^m - \omega_1\)。在实验中，我们可以设置\(\omega_L^e = \omega_2 - \omega_1\)，由此得到： \[\begin{align} \widetilde{H} = H_a + H_0 + \frac{\Omega _e}{2}\left [ \left| 2 \right\rangle \left\langle 1 \right| + \left| 1 \right\rangle \left\langle 2 \right| \right ]. \end{align}\]

离子阱中\(^{40}\mathrm{Ca}^{+}\)的动力学遵循马尔可夫过程。因此，密度算符\(\rho\)的时间演化由Lindblad主方程（\(\hbar = 1\)）描述： \[\begin{align} \dot{\rho} = -i\left[\widetilde{H},\rho \right] + L \rho L^\dagger -\frac{1}{2}\left(L^\dagger L \rho + \rho L^\dagger L \right), \end{align}\] 其中，\(L = \sqrt{\gamma} \left| 0 \right \rangle \left \langle 2 \right|\)表示从\(\left| 2 \right \rangle\)到\(\left| 0 \right\rangle\)的衰变路径，\(\gamma = 22.4\mathrm{MHz}\)为衰变速率。在此，我们旨在研究粒子的内态\(\left| 0 \right\rangle\)和\(\left| 1 \right \rangle\)。因此，可以通过绝热消去态\(\left| 2 \right\rangle\)，得到： \[\begin{align} \begin{aligned} H_{\mathrm{eff}}^{(1)} = H_a + H_0 ,\quad L_{\mathrm{eff}}^{(1)} =i \sqrt{\Gamma } \left| 0 \right\rangle \left\langle 1 \right|, \quad \Gamma = \frac{\Omega_e^2}{\gamma}, \end{aligned} \end{align}\] 其中，\(\gamma \gg \Omega_e \gg \Gamma\)确保了态\(\left| 2 \right\rangle\)的绝热消去。

接下来，我们对三束729 nm激光进行讨论：我们设\(\phi_1 = \phi_2 = 0\)，\(\delta_L^1 = -\delta_L^2 = -\omega_e\)，\(\phi_3 = \frac{\pi}{2}\)，\(\delta_L^3 = -2\omega_e\)。与第一束激光脉冲相关的哈密顿量为： \[\begin{align} H_{\mathrm{eff}1}^{(1)} = \frac{\Omega _1}{2}\mathrm{sin}[\eta(a\mathrm{e}^{-i \omega_e t} + a^\dagger \mathrm{e}^{i \omega_e t})](\sigma_+\mathrm{e}^{i \omega_e t} + \sigma_-\mathrm{e}^{-i \omega_e t}), \end{align}\] 由于\(\Omega_1/2 \ll \omega_e\)，我们可以展开正弦函数并忽略包含\(\mathrm{e}^{\pm2i\omega_e t}\)、\(\mathrm{e}^{\pm3i\omega_e t}\)等高频项，从而得到： \[\begin{align} \begin{aligned} \mathrm{sin}[\eta(a\mathrm{e}^{-i \omega_e t} + a^\dagger \mathrm{e}^{i \omega_e t})] = &\sum_{k = 0}^\infty(-n)^k\frac{\eta^{2k+1}}{(2k)!} (a\mathrm{e}^{-i \omega_e t} + a^\dagger \mathrm{e}^{i \omega_e t})\\ = & \widetilde{\eta}_1 (a\mathrm{e}^{-i \omega_e t} + a^\dagger \mathrm{e}^{i \omega_e t}), \end{aligned} \end{align}\] 其中\(n = a^\dagger a\)。再次消除高频项，我们得到： \[\begin{align} H_{\mathrm{eff}}^{(1)} = \frac{\widetilde{\eta}_1 \Omega_1}{2}(a\sigma_+ + a^\dagger \sigma_-). \end{align}\]

在\(\Gamma \gg \widetilde{\eta}_1 \Omega_1\)的条件下，态\(\left| 1 \right\rangle\)可以进一步绝热消去，得到： \[\begin{align} H_{\mathrm{eff}1}^{(2)} = 0,\quad L_{\mathrm{eff}1}^{(2)} = -\frac{\widetilde{\eta}_1 \Omega_1}{\sqrt{\Gamma}}a \left | 0 \right \rangle \left \langle 0 \right |. \end{align}\]

应用相同的方法，我们可以对另外两束729 nm激光脉冲进行操作，得到： \[\begin{align} &H_{\mathrm{eff}2}^{(2)} = 0,\quad L_{\mathrm{eff}2}^{(2)} = -\frac{\widetilde{\eta}_2 \Omega_2}{\sqrt{\Gamma}} a^\dagger \left | 0 \right \rangle \left \langle 0 \right |,\quad \\ &H_{\mathrm{eff}3}^{(2)} = 0,\quad L_{\mathrm{eff}3}^{(2)} = -\frac{\widetilde{\eta}_3 \Omega_3}{\sqrt{\Gamma}} a^2 \left | 0 \right \rangle \left \langle 0 \right |, \end{align}\] 其中，\(\widetilde{\eta}_2 = \widetilde{\eta}_1\)，\(\widetilde{\eta}_3\)为： \[\begin{align} \widetilde{\eta}_3 = \sum_{k = 1}^{\infty}(-n)^{k-1} \frac{2\eta^{2k}\Omega_3}{(k+1)!(k-1)!}. \end{align}\] 通过对自旋空间求迹，得到： \[\begin{align}\label{timecrystals} \dot{\rho} = -i[H_a,\rho] + \kappa \mathcal{D}[a]\rho + g \mathcal{D}[a^\dagger]\rho + \lambda \mathcal{D}[a^2]\rho, \end{align}\] 其中： \[\begin{equation} \left \{\begin{aligned} &H_a = -\Delta a^\dagger a + \varepsilon (a + a^\dagger),\\ &\mathcal{D}[o]\rho = o\rho o^\dagger -(o^\dagger o \rho + \rho o^\dagger o)/2,\\ &\kappa = \widetilde{\eta}_1^2 \Omega_1^2/\Gamma,\quad g = \widetilde{\eta}_2^2 \Omega_2^2/\Gamma,\quad \lambda = \widetilde{\eta}_3^2 \Omega_3^2/\Gamma. \end{aligned}\right. \end{equation}\]
